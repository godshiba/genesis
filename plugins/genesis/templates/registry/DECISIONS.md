# DECISIONS.md — Why It Is Built This Way

**G6: append-only.** Every non-obvious choice goes here with the alternatives
that were rejected and why. Never delete or rewrite an entry — if a decision is
reversed, append a new entry that supersedes it and links back.

A reader should be able to answer "why didn't they just do X?" from this file
alone.

---

## YYYY-MM-DD — <decision title>

**Decision:** What was chosen.
**Context:** The constraint or problem that forced a choice.
**Alternatives rejected:** What else was considered and the specific reason each
lost.
**Consequences:** What this commits the project to.
